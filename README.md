# IDRAKO — Brand Site

The official site of Idrako. AI director. Cinematic fantasy and sci-fi
worlds, summoned frame by frame.

Static site — no build step. Open `index.html`, or serve the folder.

## Structure

- `index.html` — one-page site: hero, brand marquee, origin, work, contact
- `css/style.css` — liquid glass, starfield, ambient color, motion
- `js/main.js` — showcase engine, proximity glow, smooth scroll
- `posters/` · `video/` — the work (brand reels + school projects)
- `assets/` — logo and wordmark

Built in silence. Remembered forever.
